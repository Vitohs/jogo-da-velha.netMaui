## APP Jogo da Velha em .NET MAUI

### Em sala de aula na Fatec Jahu, criamos o app jogo da velha, para conhecer melhor como funciona a criação de uma tela, pela primeira vez

#### Aprendemos e implementamos uma tag de marcação GRID, para conseguir criar toda a estrutura para o jogo

#### FEATURES DO APP

- GRID para criar a matriz real de um jogo da velha, dentro da contentPage
- Validações essenciais no code-behind
